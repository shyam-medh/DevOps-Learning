# DevOps-Learning

This repository collects my DevOps learning material and hands-on examples in one place.

## Folder structure

```text
tools/
  docker/           Docker projects and examples
  github-actions/   GitHub Actions course material and workflows
  jenkins/          Jenkins notes and examples
  kubernetes/       Kubernetes manifests and local cluster practice
  terraform/        Terraform configuration examples
  yaml/             YAML, JSON, and XML practice files
notes/              Small general DevOps notes and practice files
```

## Security note

Private keys, Terraform state, local Terraform plugin folders, and generated dependency folders are intentionally ignored. Do not commit cloud credentials, SSH keys, `.tfstate` files, or other secrets to this public repository.
